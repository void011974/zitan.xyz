关于云巢

云巢（CloudNest）是云朵家族自己的 SaaS 孵化器。

基于 Open SaaS 模板搭建，技术栈：

⚡ Wasp React Node.js Prisma PostgreSQL

已内置功能

✅ 用户注册 / 登录（邮箱 + 社交账号）
✅ 着陆页 + 产品展示
✅ Stripe / Polar 支付
✅ 后台管理面板
✅ 数据分析
✅ 文件上传（S3）
✅ AI 功能模块
✅ 博客 + 文档系统

🚧 目前正在紧锣密鼓建设中，敬请期待…
